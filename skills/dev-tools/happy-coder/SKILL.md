# Happy Coder — Mobile/Web Client for Claude Code

## Tool Info

- **CLI**: `happy-coder` (installed globally, v0.12.0)
- **Package**: `happy-coder`
- **Purpose**: Mobile and web client for remote Claude Code/Codex access

## When to Use

User mentions: mobile Claude Code, phone coding, remote Claude, Claude on phone/tablet, happy coder, mobile access.

## Quick Start

```bash
happy-coder
```

## Setup

### First Time
```bash
happy-coder start
```

### Connect from Mobile
Scan the QR code shown in terminal, or open the URL in mobile browser.

## Features

- Mobile-friendly Claude Code interface
- Web client for browser access
- Remote session management
- QR code pairing for quick mobile connection

## Notes

- Keeps Claude Code session running on host machine
- Mobile device connects via local network or tunnel
- Works with Claude Code and Codex backends
