# OpenCLI — Website to CLI Converter

## Tool Info

- **CLI**: `opencli` (installed globally, v1.8.3)
- **Package**: `@jackwener/opencli`
- **Capability**: AI-powered — turns any website or Electron app into a CLI

## When to Use

User mentions: turn website into CLI, automate web, CLI for web app, web to terminal, OpenCLI.

## Quick Start

```bash
opencli
```

## Common Workflows

### Turn a Website into CLI
```bash
opencli <url>
```

### Turn an Electron App into CLI
```bash
opencli --electron <app-path>
```

## Key Features

- AI-powered element detection and interaction
- Works with any website (login, forms, search, etc.)
- Electron app CLI wrapping
- Scriptable — pipe output to other commands
